### How to Use

You have a few options to get started with BATman scripts:

1.  **Explore Script Folders:**
    * Find the tool you're interested in within the "Available Scripts" table.
    * Click the "**Go to Folder**" link to view its contents directly on GitHub, including its dedicated `README.md` for detailed instructions.

2.  **Download Individual Scripts (Recommended for quick use):**
    * Find the script you need in the "Available Scripts" table.
    * Click the "**Download ZIP**" link. This will take you to the DownGit website in your current browser tab.
    * On the DownGit page, **click the 'Download' button** to initiate the `.zip` file download for that specific folder.
    * Extract the downloaded `.zip` file and navigate into the folder.

3.  **Clone the Entire Repository:**
    If you want the full collection and prefer to keep it updated with Git:

    ```bash
    git clone [https://github.com/xvacorx/BATman.git](https://github.com/xvacorx/BATman.git)
    ```

    Alternatively, you can download the entire repository as a **ZIP file** directly from the main GitHub page by clicking the green "Code" button and selecting "Download ZIP."

4.  **Run the Script:**
    Once you have the script's folder, simply double-click the `.bat` file inside it. Most advanced scripts (like the Toolbox) feature intelligent auto-elevation and will automatically request administrative privileges if required.

**Important Note:** Always review the `README.md` file inside each specific script's folder for detailed operational instructions.
